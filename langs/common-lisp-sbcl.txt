to emacs:

m = modifier key (alt, mac)

m-x
  - command prompt
  - lisp-mode, markdown-mode, etc.
    - called a "major-mode"
  - abbrev
    - autocomplete
	- m-/
  - abbrev-with-popup-window
  - describe key (c-x ?)
  - describe bindings
  - documentation-describe-bindings
    - list all bindings

c-x
  - (d)irectory viewer
  - (f)ile
  - (s)ave
  - o / m-o
    - switch windows
  - (b)uffer
  - left/right arrows
    - left/right buffer
  - (c)lose
    - quit
  - (w)rite
  - ?
    - describe-key

m
  - / (or c-p in vi's insert-mode)
